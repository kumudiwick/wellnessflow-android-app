# 🌿 WellnessFlow – Android Wellbeing Tracker

WellnessFlow is an Android mobile application designed to help users track their daily wellbeing, monitor moods, and build healthy habits through a simple and intuitive interface.

---

## 📱 Features

- 😊 Mood tracking with emoji-based selection
- 📊 Daily wellbeing logging
- 📝 Habit and activity tracking
- 📈 Data visualization to identify trends
- 🎯 Clean and user-friendly interface

---

## 🛠 Technologies Used

- Kotlin
- Android Studio
- SQLite (Local Database)
- XML (UI Design)

---

## 🧠 Key Learning Outcomes

- Built a complete Android application from scratch  
- Implemented local data storage using SQLite  
- Designed responsive UI using XML layouts  
- Applied MVVM-style project structure
